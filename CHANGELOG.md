# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-02-15

### Added

- Passwordless authentication with OTP-based login
- Gmail OAuth 2.0 integration for connecting email accounts
- API key management (create, list, update, delete) with configurable expiry
- Interactive Swagger UI API documentation
- AWS CDK infrastructure (Lambda, API Gateway, DynamoDB, IAM)
- CI/CD with GitHub Actions (lint on PR, deploy on push to main)
- Biome linting and formatting with pre-commit hooks
